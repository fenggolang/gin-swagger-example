#### swagger yaml编写规则
```yaml
in: query //参数的位置
name: limit //参数的名字
type: integer //类型
required: false //是否必须
default: 20 //默认
minimum: 1 //最小值
maximum: 100 //最大值
description: The numbers of items to return. //参数的描述
```
